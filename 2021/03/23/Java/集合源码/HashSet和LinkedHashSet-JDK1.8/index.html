总结

这两个 Set 集合实质上都一个 Map，其所有操作都是基于这个 map，所以它们的特性跟 HashMap 基本一样。

所以 HashSet 检查元素重复的过程是：

  • 计算元素的 hash 值；
  • 计算对应的桶索引；
  • 将元素跟桶上的元素比较：
    • 若 hash 值不同，则判断两个元素不同
    • 若 hash 值相同，则继续比较，
      • 若 == 或 equals() 比较返回 true，则认为相同；
      • 否则认为不同。

即两个元素相同的标准是：hash 值相同，且 == 或 equals() 比较仍相同。
