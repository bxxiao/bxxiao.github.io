总结

  • 根据 搜索区间 来确定while循环中是 < 还是 <= 。
  • 同样根据搜索区间决定 mid 。
  • 分析二分查找代码时，不要出现 else，全部展开成 else if 方便理解。
  • 注意「搜索区间」和 while 的终止条件，如果存在漏掉的元素，记得在最后检查。
  • 如需要搜索左右边界，只要在 nums[mid] == target 时做修改即可。搜索右侧时需要减一。
