*MySQL5. 和 8.0.22 都可以照这个安装方法，这里以 5.7.30 为例**。

使用 installer 只能安装在 c 盘 。

  • 解压，重命名。

  • 启动 MySQL 服务：

  • 输入mysql -u root -p

  • 输入上述随机生成的密码进入数据库。进入后需要修改密码，因为随机生成的密码在登录后会过期。（这时若不修改密码，执行一切sql语句都会报错，提醒修改密码）

  • 修改密码的命令：

    • alter user 'root'@'localhost' identified by 'newPassword';

    • flush privileges;
